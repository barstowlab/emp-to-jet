2*Acetyl-CoA → Acetoacetyl-CoA + CoA
Acetoacetyl-CoA + Acetyl-CoA + H2O → HMG-CoA + CoA
HMG-CoA + 2*NADPH + 2*Hplus → Mevalonate + CoA + 2*NADPplus
Mevalonate + ATP → ADP + Mevalonate-5-phosphate
ATP + Mevalonate-5-phosphate → ADP + Mevalonate-5-diphosphate
ATP + Mevalonate-5-diphosphate → ADP + Isopentanyl diphosphate + Orthophosphate + CO2
Isopentanyl diphosphate → Dimethylallyl diphosphate
Isopentanyl diphosphate + Dimethylallyl diphosphate → Geranyl diphosphate + diphosphate
Geranyl diphosphate + H2O → Geraniol + diphosphate
NADH + NADPplus → NADPH + NADplus
