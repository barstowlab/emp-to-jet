CO2 + H2O → HCO3- + Hplus
ATP + Acetyl-CoA + HCO3- → ADP + Orthophosphate + Malonyl-CoA
Acetyl-CoA + Malonyl-CoA → Acetoacetyl-CoA + CO2 + CoA
Acetoacetyl-CoA + NADPH + Hplus → 3R-3-Hydroxybutanoyl-CoA + NADPplus
3R-3-Hydroxybutanoyl-CoA → But-2-enoyl-CoA + H2O
But-2-enoyl-CoA + NADPH + Hplus → Butanoyl-CoA + NADPplus
Malonyl-CoA + Butanoyl-CoA → CoA + CO2 + 3-Oxohexanoyl-CoA
3-Oxohexanoyl-CoA + NADH + Hplus → S-Hydroxyhexanoyl-CoA + NADplus
S-Hydroxyhexanoyl-CoA → trans-hex-2-enoyl-CoA + H2O
trans-hex-2-enoyl-CoA + NADPH + Hplus → Hexanoyl-CoA + NADPplus
Hexanoyl-CoA + NADH + Hplus → CoA + Pentanal + NADplus
Pentanal + Oxygen + 2*NADPH + 2*Hplus → Pentane + Formate + H2O + 2*NADPplus
Formate → CO2 + NADH + Hplus
NADH + NADPplus → NADPH + NADplus
