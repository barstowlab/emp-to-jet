CO2 + H2O → HCO3- + Hplus
ATP + Acetyl-CoA + HCO3- → ADP + Orthophosphate + Malonyl-CoA
Propionyl-CoA + Malonyl-CoA → 5-Acetoacetyl-CoA + CO2 + CoA
5-Acetoacetyl-CoA + NADPH + Hplus → 3R-3-Hydroxypentanoyl-CoA + NADPplus
3R-3-Hydroxypentanoyl-CoA → Pent-2-enoyl-CoA + H2O
Pent-2-enoyl-CoA + NADPH + Hplus → Pentanoyl-CoA + NADPplus
Malonyl-CoA + Pentanoyl-CoA → CoA + CO2 + 3-Oxoheptanoyl-CoA
3-Oxoheptanoyl-CoA + NADH + Hplus → S-Hydroxyheptanoyl-CoA + NADplus
S-Hydroxyheptanoyl-CoA → trans-hept-2-enoyl-CoA + H2O
trans-hept-2-enoyl-CoA + NADPH + Hplus → Heptanoyl-CoA + NADPplus
Malonyl-CoA + Heptanoyl-CoA → CoA + CO2 + 3-Oxononanoyl-CoA
3-Oxononanoyl-CoA + NADH + Hplus → S-Hydroxynonanoyl-CoA + NADplus
S-Hydroxynonanoyl-CoA → trans-non-2-enoyl-CoA + H2O
trans-non-2-enoyl-CoA + NADPH + Hplus → Nonanoyl-CoA + NADPplus
Malonyl-CoA + Nonanoyl-CoA → CoA + CO2 + 3-Oxoundecanoyl-CoA
3-Oxoundecanoyl-CoA + NADH + Hplus → S-Hydroxyundecanoyl-CoA + NADplus
S-Hydroxyundecanoyl-CoA → trans-undec-2-enoyl-CoA + H2O
trans-undec-2-enoyl-CoA + NADPH + Hplus → Undecanoyl-CoA + NADPplus
Malonyl-CoA + Undecanoyl-CoA → CoA + CO2 + 3-Oxotridecanoyl-CoA
3-Oxotridecanoyl-CoA + NADH + Hplus → S-Hydroxytridecanoyl-CoA + NADplus
S-Hydroxytridecanoyl-CoA → trans-tridec-2-enoyl-CoA + H2O
trans-tridec-2-enoyl-CoA + NADPH + Hplus → Tridecanoyl-CoA + NADPplus
Malonyl-CoA + Tridecanoyl-CoA → CoA + CO2 + 3-Oxopentadecanoyl-CoA
3-Oxopentadecanoyl-CoA + NADH + Hplus → S-Hydroxypentadecanoyl-CoA + NADplus
S-Hydroxypentadecanoyl-CoA → trans-pentadec-2-enoyl-CoA + H2O
trans-pentadec-2-enoyl-CoA + NADPH + Hplus → Pentadecanoyl-CoA + NADPplus
Malonyl-CoA + Pentadecanoyl-CoA → CoA + CO2 + 3-Oxoheptadecanoyl-CoA
3-Oxoheptadecanoyl-CoA + NADH + Hplus → S-Hydroxyheptadecanoyl-CoA + NADplus
S-Hydroxyheptadecanoyl-CoA → trans-heptadec-2-enoyl-CoA + H2O
trans-heptadec-2-enoyl-CoA + NADPH + Hplus → Heptadecanoyl-CoA + NADPplus
Heptadecanoyl-CoA + NADH + Hplus → CoA + Heptadecanal + NADplus
Heptadecanal + Oxygen + 2*NADPH + 2*Hplus → Hexadecane + Formate + H2O + 2*NADPplus
Formate → CO2 + NADH + Hplus
NADH + NADPplus → NADPH + NADplus
